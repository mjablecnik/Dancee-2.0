# Dancee App

A Flutter mobile application for discovering and managing dance events and courses.

## Requirements

- [Flutter](https://docs.flutter.dev/get-started/install) 3.x (Dart SDK ^3.6.2)
- Flutter-supported platform: Android, iOS, Web, Linux, macOS, or Windows

Verify your Flutter installation:

```bash
flutter doctor
```

## Getting Started

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

To target a specific device:

```bash
flutter run -d <device-id>   # e.g. -d chrome, -d linux, -d emulator-5554
flutter devices              # list available devices
```

### Build

```bash
# Android APK
flutter build apk

# Web
flutter build web

# Linux
flutter build linux
```

### Run tests

```bash
flutter test
```

### Lint

```bash
flutter analyze
```

## Code generation

Localization strings in `lib/i18n/` are generated by [slang](https://pub.dev/packages/slang). If you add or change translation files (`assets/i18n/*.i18n.json`), regenerate the code with:

```bash
dart run build_runner build --delete-conflicting-outputs
```

## Project structure

```
lib/
  core/          # Theme and global styles
  data/          # Data repositories (events, courses, user, ...)
  i18n/          # Generated localization code
  screens/       # Feature screens (auth, events, courses, profile, saved)
  shared/        # Reusable widgets and UI elements
```

Routing is handled by [go_router](https://pub.dev/packages/go_router). State management uses [flutter_bloc](https://pub.dev/packages/flutter_bloc). Multi-language support (EN/CS) is powered by [slang](https://pub.dev/packages/slang_flutter).
